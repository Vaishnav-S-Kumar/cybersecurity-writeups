# Binary Digits

## Description

This file doesn't look like much... just a bunch of 1s and 0s. But maybe it's not just random noise. Can you recover anything meaningful from this?

## Solution

Download the file, using the ```cat``` command view the contents of the file. Since it consist of binary data (only 0's and 1's), it can be converted. Using cyberchef.io, input the file and using the binary option convert the file. After conversion, the output has ```JFIF``` which means it is a image file. Thus, use the render image option to convert the latest output to image.

The flag is shown in the image.
