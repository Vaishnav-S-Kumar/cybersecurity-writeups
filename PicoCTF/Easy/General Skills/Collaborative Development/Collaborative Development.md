# Collaborative Development

## Description

My team has been working very hard on new features for our flag printing program! I wonder how they'll work together?

## Solution 

Download the files. After unzipping the files, there is a single python file and also single commit entry in the log (which was displayd using ```git log``` command)

According to the description, it means the multiple people are working on this. For this branches are required to prevent collision. To check how many branches aare there, use the command ```git branch```

To change the branch for viewing the different development, use the command ```git checkout <branch-name>``` and check the changes in the file. (Check all the branches)
