# 13

## Description

Cryptography can be easy, do you know what ROT13 is? cvpbPGS{abg_gbb_onq_bs_n_ceboyrz} 

## Solution

The method used here is called ROT13, where the characters are changed with the character that appear after 13th character. i.e if I use ROT13 on "abcd", the output would be "nopq"

To convert this cipher text, use the ```tr``` command: 
```
echo <cipher> | tr a-zA-Z n-za-m N-ZA-M
```
or else you can also use cyberchef.io and use the option "ROT13"
