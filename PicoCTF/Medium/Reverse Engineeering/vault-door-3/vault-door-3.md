## Solution

Download the source code from the hyperlink provided in the description. The password is not directly given in the code but a string is given which is the manipulated version of the original password which is processed in a way in which the characters are scrambled and is placed at different position using for loops. 

To find the original password, a python or java program can be created to reverse the string given in the code, By which the original password appears. Thus, by adding password between ```{ }``` in ```picoCTF{}``` is the flag for the challenge.
