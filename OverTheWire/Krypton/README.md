# OverTheWire Krypton 
## This platform is used for learning basic cryptography

This is designed for beginners to understand how classical encryptoon techniques work. To progress through the levels players must decrypt or break the given ciphertext to find the password of the first level.

Level 0 does not require SSH instead an encoded string is given, which should be decoded to find the password to level 1.

The format for SSH is:
```
ssh kryptonX@krypton.labs.overthewire.org -p 2231
```
where X stands for the level number
