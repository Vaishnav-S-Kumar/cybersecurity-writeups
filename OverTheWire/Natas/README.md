# OverTheWire Natas

## Platform to learn the basics of the serverside web-security

This directory contains the walkthrough and explainations for each challenge of Natas Wargame

Natas focuses on web security vulnerabilities and helps learners understand how common web attacks work in a controlled learning environment.

Each level demonstrates a different web vulnerability and teaches how improper implementations can lead to security issues.

The tools needed to complete the natas challenge is:
```
Web Browser
Burpsuite
curl
ZAP Proxy
```

To access each level enter the URL Format given below
```
Username: natasX
Password: **************
URL:      http://natasX.natas.labs.overthewire.org
```
X represents the level number and each level the objective is to find the password of the next level
