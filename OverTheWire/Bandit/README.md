# OverTheWire Bandit 
## This was created for learning linux in an interactive way

Bandit is an online platform aimed for absolute beginners for learning linux. They use ssh to connect to each level of the challenge and to advance to next level requires the user to find the password using different linux commands. 

The best resource to learn about each tool is the man command which provides the manual of a command. for example "man ls" provides the manual of ls command.

Each level of bandit is connected by the command 
```
ssh banditX@bandit.labs.overthewire.org -p 2220 
```
where X stands for the level number (currently 0-34)

To complete each level, the player has to find the password using the description/instruction provided in the website about the level. This password.

**Level 0 can be accessed by the password bandit0**
